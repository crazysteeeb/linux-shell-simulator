# Linux Shell Simulator

A custom Linux shell implementation built in C++ demonstrating command execution and operating systems concepts.

## Overview

This project simulates core shell functionality commonly found in Linux command-line environments.

The shell demonstrates:
- Command parsing
- Process execution
- Linux terminal interaction
- Systems programming concepts

## Features

- Command-line input handling
- Shell command parsing
- Process execution logic
- Linux terminal behavior
- Interactive command processing

## Technologies Used

- C++
- Linux
- Visual Studio Code

## Concepts Demonstrated

- Process management
- Command execution
- Shell architecture
- Linux systems programming
- Operating systems fundamentals

## How to Compile

```bash
g++ newshell.cpp -o shell
```

## How to Run

```bash
./shell
```
